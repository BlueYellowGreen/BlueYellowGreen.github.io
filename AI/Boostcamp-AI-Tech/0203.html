#matplotlib #bar_plot #line_plot #scatter_plot

  시각화하는 방법에 배우기에 앞서 기본적인 개념을 알아야 한다.

# 데이터셋의 종류

  • 정형 데이터   ➜   tabular 데이터에 적합하다. 시각화에 가장 쉬운 데이터 종류이며, 주로 통계적 특성 및 feature 사이 관계를 알아본다.

  • 시계열 데이터   ➜   정형 데이터와 비정형 데이터 모두 존재하며, 주로 추세, 계절성, 주기성 등을 살핀다.

  • 지리/지도 데이터   ➜   지도 정보와 원하는 정보간의 조화가 중요하다.

  • 관계형(네트워크) 데이터   ➜   관계를 시각화하는 방식으로, 주로 객체를 Node, 그리고 관계를 Link 로 나타낸다.

  • 계층적 데이터   ➜   관계 중에서도 포함관계가 분명한 데이터이다. (Tree, Treemap, Sunburst)


# 데이터의 종류

  • 수치형 ( 연속형 vs. 이산형 )

  • 범주형 ( 명목형 vs. 순서형 )


# 전주의적 속성

  Pre-attentive Attribute 로, 주의를 주지 않아도 인지하게 되는 요소이다. 회색 점들로 가득찬 공간에 빨간 점 하나가 눈에 잘 보이듯이, 적절하게 이러한 속성을 이용하면 시각적 분리에 유리하다. (언급한 것처럼 색채/색조 대비 방식을 자주 사용한다.)


# Matplotlib

  이제 python 의 대표적인 시각화 라이브러리인 Matplotlib 에 대해서 학습하였다. 주로 import matplotlib.pyplot as plt 방식으로 라이브러리를 불러와 사용한다. 사용방식은 Figure 라는 큰 도화지에 Ax 라는 subplot 을 1개 이상 추가하며 그려낸다. Ax 에는 여러개의 그래프를 동시에 그릴 수 있다.

  추가하는 방식도 여러가지가 있지만, 주로 .add_subplot(index) 방식으로 위치를 지정하며 추가하는 방식을 사용한다.

fig = plt.figure()
ax1 = fig.add_subplot(1, 2, 1) # ■ □
ax2 = fig.add_subplot(1, 2, 2) # □ ■

plt.show()

fig = plt.figure()

# ■
# □
ax1 = fig.add_subplot(2, 1, 1)

# □
# ■
ax2 = fig.add_subplot(2, 1, 2)

plt.show()

.add_subplot(121) 같은 방식도 가능하지만, 추후 수십 수백개의 features 에 대해서 for 문으로 Ax 를 추가하려면 먼저 언급한 방식이 편리하다.


# 기본적인 파라미터와 함수

  기본적인 속성에 대해 알아보자.

  1. label parameter
    한 Ax 에 label='string' 속성을 추가하면, 나중에 legend() 로 해당 string 으로 범례를 보여줄 수 있다.

  2. .set_title('string')
    한 Ax 에 대해 가장 큰 제목을 추가한다. 반대로 해당 제목 데이터를 얻으려면 .get_title() 을 사용하면 된다.

  3. .set_xticks([1, 2, ...])
    한 Ax 에 대해 X 축의 범위를 정의한다. 빈 리스트로 입력하면 눈금이 사라진다.

  4. .set_xticklabels(['zero', 'one', ...])
    한 Ax 에 대해 X 축 눈금의 Text 를 어떻게 표현하는지 정의한다.

  5. .annotate(text='sentence', xy=(1, 2))   or   .annotate(text='sentence', xy=(1, 2), arrowprops=dict(facecolor='black'))
    한 Ax 에 대해 Text 를 지정한 위치에 그린다. 주로 화살표와 같이 특정 데이터를 설명하는데 사용한다.


# Bar Plot

  직사각형 막대를 사용하여 데이터의 값을 표현하는 차트/그래프로, 범주 (category)에 따른 값을 비교하기에 적합하다. 막대가 수직 ➜ .bar()   /   막대가 수평 ➜ .barh() 방식을 사용한다. barplot에도 여러 방식이 존재한다. 동일한 Ax 에 .bar() 를 누적하여 그리는 stack 방식, 직사각형 속 비율로 표시하는 percentage 방식(text로 수치를 나타내면 더욱 좋다.), stack 처럼 누적해서 그리지만 alpha=0.7 처럼 투명도를 조절하여 동시에 보여주는 overlaped 방식, 그리고 한 tick에 겹치지 않고 연달아 보여주는 grouped 방식이 있다. 이 모두 seaborn 을 이용하면 간편하게 구현할 수 있다.

  Bar Plot 에서 권장 사항이 존재한다. X 축의 시작은 반드시 0이어야 한다는 점과 잉크 양 비례 법칙에 따라 큰 값의 데이터는 더 많이 표현해야 한다. 그리고 정렬은 필수적이며, 정렬 방식은 시계열은 시간순으로, 수치형은 크기순으로, 순서형은 범주의 순서대로, 그리고 명목형은 범주의 값에 따라 정렬한다.


# Line Plot

  꺾은선 그래프라고도 불리는 Line Plot 은 시간/순서에 대한 변화에 적합하며, 추세를 살펴보기 위해 사용한다. 이름과는 다르게 .plot() 으로 그래프를 그린다. 그리고 가독성을 높이기 위해 한 Ax 에 사용되는 선은 5개 이하로 제한한다.


# Scatter Plot

  이론상 5차원까지 표현 가능(color, marker, size)한 Scatter Plot 은 상관 관계을 확인하기 위해 사용한다. 다만, 상관관계를 가진다고 해서 인과관계에 있는 것은 아니라는 점이다. Scatter Plot 을 통해 군집, 값 사이의 차이, 그리고 이상치를 파악한다.
