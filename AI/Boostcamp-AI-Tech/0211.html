# 2022/02/11, 새로 알게된 점

#polar_coordinate #pie_charts #missingno #squarify #pywaffle #pyvenn

  오늘은 저번주 Data Visualizatoin 에 이어서 학습한다. Polar coordinate 에서는 어떻게 시각화를 진행하는지 알아보자.

일반적으로 Polor coordinate 는 잘 사용하지 않지만, 대표적인 두 가지 Polar Plot 과 Radar Plot 을 알아볼 것이다.


# Polar Plot

  극좌표계를 사용하는 것을 Polar Plot 이라고 한다. 극좌표계는 중심의 거리 과 각도 를 사용한다. 회전 이나 주기성 등을 표현하기에 적합하다. Matplotlib 에서는 단순히 projection = polar 를 추가해서 사용할 수 있는데, 이는 기존에 사용했던 Line, Bar, Scatter 등 모두 적용가능하다는 뜻이다.

fig = plt.figure()
ax = fig.add_subplot(111, projection='polar')
# ax = fig.add_subplot(111, polar=True) # polar=True 방식으로도 가능하다.
plt.show()

위 코드의 결과로, 다음의 그래프를 얻을 수 있다.


그리고 Polar Coordinate 를 조정하는 여러 함수가 존재한다.

  • .set_rmax() / .set_rmin()   ➜   반지름 조정
    • .set_rmin() 을 지정한다 하더라도 부채꼴 모양이 되는 것이 아니라, 시작 tick 이 바뀌는 개념이다.
  • .set_rticks(list)   ➜   리스트 값 대로 tick 범위가 형성된다.
  • .set_rlabel_position   ➜   tick 이 적히는 위치(각도)를 조정한다.
  • .set_thetamin() / .set_thetamax   ➜   부채꼴 모양을 사용하게 해준다. 지정한 각도 범위만큼 생성된다.
    • 다만, 부채꼴 모양을 사용하면 tick 위치(각도)는 변경이 안되는 것으로 보인다.
# set tick position
fig = plt.figure()
ax = fig.add_subplot(111, polar=True)

ax.set_rlabel_position(-90)  
plt.show()


# circular sector form
fig = plt.figure()
ax = fig.add_subplot(111, polar=True)

ax.set_thetamin(45)
ax.set_thetamax(135)
plt.show()


scatter, bar, line, fill 모두 projection = 'polar' 를 통해 polar plot 으로 표현할 수 있다.


물론 꼭 극좌표계로 그릴 필요는 없다. 직교 좌표계 , 를 각각 , 처럼 변환해서 사용해도 된다.


# Radar Plot

  Polar Plot 을 좀 더 잘 사용하기 위해서 Radar Plot 을 사용하는 방법이 있다. 극좌표계를 사용하는 대표적인 차트로, 중심점을 기준으로 N개의 변수 값을 표현할 수 있다. 이 모양이 마치 별 같아 Star Plot 으로 불리기도 한다. 게임 속 캐릭터의 능력치 표현 등에 자주 사용되는 방식으로 익숙한 모양이다.

다만 사용함에 있어서 주의해야 하는 점들이 있는데, 우선 각 feature 들이 독립적이고 척도가 같아야 한다. 그리고 데이터 속에 순서형 변수와 수치형 변수가 함께 있다면 고민을 해야 한다. 그리고 Radar Chart 속 다각형의 면적은 큰 의미를 가지지 않는다. 이는 feature 순서에 따라 달라질 수 있기 때문이다. 그리고 feature 가 많아질수록 가독성이 떨어진다.

Radar Plot 을 커스텀하는 함수도 존재한다.

  • .set_thetagrids()   ➜   각도에 따른 그리드 및 tick label 을 변경한다.
  • .set_theta_offset()   ➜   시작 각도를 변경한다.

Radar Plot 을 사용하기 위해 N 개의 feature 를 사용할 지 정한 뒤, 각각의 feature 위치에 맞게 데이터 값이 적용되도록 theta 에 데이터 값을 넣는다. 한 가지 팁으로는, 이어지는 모양의 다각형으로 그리기 위해서 첫 데이터를 마지막에 또 추가시켜줘야 한다. 그런 다음 시작 각도 및 feature tick label 변경하면 다음과 같다.

fig = plt.figure(figsize=(4, 4))
ax = fig.add_subplot(111, projection='polar')

values = [...data...]                              # N개의 데이터. 이 예제에서는 N=6
values.append(values[0])                           # 이어지는 N각 형을 위해 마지막에 첫 데이터 추가

ax.plot(theta, values)
ax.fill(theta, values, alpha=0.5)                  # 내부가 채워진 형태

ax.set_thetagrids([n*60 for n in range(6)], stats) # 360도 / 6개 = 60도 마다 그리드를 그림
ax.set_theta_offset(np.pi/2)                       # 첫 tick label 의 위치를 90도로 변경

plt.show()

# Pie Charts

  극좌표계와 약간 유사한 Pie Chart 및 Pie Chart 변형에 대해 살펴보자. 내용을 짧지만, 많이 사용되는 시각화이다.

Pie Chart 는 원을 부채꼴로 분할하여 표현하는 통계 차트로, 전체를 백분위로 나타낼 때 유용하다. 많이 사용되는 차트이지만, feature 간 비교가 어려워서 지양하는 것이 좋다. 그래서 차라리 비교를 위한다면 bar plot 이 좋다 (혹은 pie chart 를 써야 한다면 bar plot 과 같이 보여주자). 혹은 나중에 설명할 waffle chart 가 더 비교에 유용하다.

Pie Chart 를 커스텀하기 위한 파라미터는 다음과 같다. ( ax.pie(data, ...) )

  • labels=   ➜   각각의 조각에 대해 어떤 label 을 보여줄지 정하는 것으로, 리스트 등으로 입력한다.
  • labeldistance=   ➜   각각의 label 이 조각으로부터 얼마나 떨어져서 그려지는지 정한다.
  • rotatelabels=   ➜   각도를 입력하는데, 해당 각도만큼 각각의 label 들이 회전해서 그려진다.
  • startangle=   ➜   해당 각도를 기준로 반시계 방향으로 데이터 값 비율을 그려간다.
  • explode=   ➜   one-hot vector (리스트) 형태로 한 조각 튀어나와 있는 형태의 효과를 주려는 데이터에 대해서만 튀어나온 거리를 설정한다.
    • ex) data=[10, 20, 5, 40], ax.pie(..., explod=[0, 0, 0.3, 0], ...)
  • shadow=   ➜   True / False 로 그림자 유무를 설정한다.
  • autopct=   ➜   각각의 조각에 비율을 표시하는데, 어떤 format 으로 표시하는지 지정한다. ( autopct='%1.1f%%' )
  • pctdistance=   ➜   표시되는 비율을 원의 중심으로부터 얼마나 멀어질지 지정한다.
  • textprops=   ➜   표시되는 비율(텍스트)의 색상을 지정한다.
  • counterclock=   ➜   True / False 로 시계/반시계 방향으로 그릴지 지정한다.
  • radius=   ➜   Pie Chart 자체의 크기를 지정한다.

Donut chart

  중간이 비어있는 Pie Chart 로써, 디자인적으로 선호되는 편이다. 인포그래픽에서 종종 사용되며, plotly에서 쉽게 사용할 수 있다. 하지만 기존 Pie Chart 보다 가독성이 떨어져 비교하기 어렵기 때문에, 데이터 분석 시에 사용은 지양하는 것이 좋다.

Matplotlib 으로 구현 시, patch 라는 트릭을 사용하여 donut chart 를 구현한다. 동일하게 Pie Chart 를 그린 뒤, 중심을 기준으로 배경색과 같은 원을 그린다.

fig, ax = plt.subplots(1, 1, figsize=(7, 7))

ax.pie(data, labels=labels, startangle=90, autopct='%1.1f%%')

centre_circle = plt.Circle((0, 0), 0.7, fc='white') # 배경색과 같은 원을 그려
ax.add_artist(centre_circle)                        # Pie Chart 를 덮는다.

plt.show()

Sunburst Chart

계층적 데이터를 시각화하는데 사용된다. 구현 난이도에 비해 화려하다는 장점이 있지만, Treemap 을 사용하는 것이 더 가독성에 좋다. Sunbusrt Chart 또한 Plotly 로 쉽게 사용 가능하다.


# 다양한 시각화 라이브러리

  1. missingno

  결측치를 시각화하는 라이브러리이다. 빠르게 결측 치의 분포를 확인하고 싶을 때 사용하면 좋다. Python 에서 pip install missingno 로 설치할 수 있다. 그리고 일반적으로 Null 값에 대한 분포를 보여주는 방법과, Bar Plot을 통해 Null 값의 분포를 보여주는 방식이 있다.

import missingno as msno
msno.matrix(data)
# msno.matrix(data, sort='descending') # row 당 결측치의 개수가 달라 정렬해서 볼 수도 있다.

msno.bar(data) # bar chart 로 직접적 개수 확인

  1. Treemap

  계층적 데이터를 표현하는 대표적인 시각화 방법이다. 사각형을 분할하는 타일링 알고리즘에 따라 형태가 달라지는데, 수직으로만 분리되면 가독성에 좋지 않기에 적절한 방법을 사용해야 한다. 그리고 Treemap 은 큰 사각형을 분할하여 전체를 나타내는 모자이크 플롯(Mosaic plot)과도 유사하다. Python 에서는 2가지 라이브러리가 있다 ( squarify, Plotly ). squarify 는 pip install squarify 를 통해 설치할 수 있다.

import squarify
squarify.plot(data, label=label, color=color, pad=pad, text_kwargs={...}, ax=ax)
  • label=   ➜   각 영역마다 text label 을 지정한다.
  • color=   ➜   색을 개별적으로 지정 가능하다.
  • pad=   ➜   여백을 지정한다.
  • text_kwargs=   ➜   텍스트 요소를 딕셔너리로 전달할 수 있다.

  1. Waffle Chart

  와플 형태로 discrete 하게 값을 나타내는 차트이다. 기본적인 형태는 정사각형이지만, 원하는 벡터 이미지로도 사용할 수 있다. Icon 을 사용한 Waffle Chart (Pictogram Chart) 도 가능해서 인포그래픽에서 유용하다. Python 에서 pip install pywaffle 을 통해 설치할 수 있다.

from pywaffle import Waffle
fig = plt.figure(FigureClass=Waffle, row=5, columns=10, values=data, figsize=(5, 3))
plt.show()
  • row 와 columns 로 사각형의 형태를 지정하며, values 로 데이터를 전달한다.
  • legend=   ➜   딕셔너리로 loc, bbox_to_anchor 등의 값을 전달한다.
  • cmap_name=   ➜   컬러맵을 전달할 수 있다.
  • colors=   ➜   각 범주의 색을 전달할 수 있다.
  • starting_location=   ➜   네 꼭지점을 기준으로 시작점을 잡을 수 있다 ( SE, NW, SW, NE ).
  • vertical=   ➜   True / False 로 세로로 진행하는지 여부를 지정한다.
  • block_arranging_style=   ➜   어떤 식으로 나열할지 정할 수 이다 (default=snake / new-line).
  • Icon - Font Awesome 의 아이콘을 사용할 수 있다.
    • icons=   ➜   아이콘 명칭
    • icon_legend=   ➜   아이콘 범례 사용 여부
    • font_size=   ➜   아이콘 사이즈

  1. Venn

  집합(set) 등에서 사용하는 벤 다이어그램을 표현할 수 있다. EDA 보다는 출판 및 프레젠테이션에 주로 사용되지만, 디테일한 사용은 힘들다. Python 에서는 pip install pyvenn 이나 pip install matplotlib-venn 로 설치할 수 있다. 둘 다 matplotlib 기반이라 문법도 비슷하다.

이진법을 사용하여 각각에 들어갈 값을 지정할 수 있다.

  • 01   ➜   1번째 Set에 들어갈 내용
  • 02   ➜   2번째 Set에 들어갈 내용
  • 03   ➜   교집합에 들어갈 내용
from matplotlib_venn import venn2
venn2(subsets=(3, 2, 1))
