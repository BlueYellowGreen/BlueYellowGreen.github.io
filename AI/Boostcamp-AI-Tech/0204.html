# 2022/02/04, 새로 알게된 점

#Text #Color #Facet #Seaborn

  세부적인 사용 방법에 대해 알아보자.

# Text

  • fig.suptitle('string')   ➜   figure 의 title 을 설정한다.

  • ax.set_title('string')   ➜   Ax 의 title 을 설정한다.

  • ax.set_xlabel('string')   ➜   Ax 의 X 축 title 을 설정한다.

  • ax.set_ylabel('string')   ➜   AX 의 Y 축 title 을 설정한다.

  • fig.text(x, y, s='string')   ➜   figure 기준으로 x, y 위치(비율)에 text 를 설정한다.

  • ax.text(x, y, s='string')   ➜   Ax 기준으로 x, y 위치(tick)에 text 를 설정한다.


  • family=   ➜   폰트 ( serif, sans-serif, fantasy, monospace, ... )

  • style= or fontstyle=   ➜   폰트 스타일 ( normal, italic, bold italic, ... )

  • weight= or fontweight=   ➜   굵기 ( light, normal, medium, semibold, bold, heavy, black, ... )

  • size= or fontsize=   ➜   크기. 숫자를 입력하거나 정해진 크기 단위 입력 ( xx-small, x-small, small, medium, large, x-large, xx-large, ...)

  • color=   ➜   웹에서 사용하는 color 코드, 이름 사용 가능

  • linespacing=   ➜   줄 간격

  • backgroundcolor=   ➜   배경 색

  • alpha=   ➜   투명도

  • zorder=   ➜   z 축. 보여지는 순서

  • visible=   ➜   보여지는지에 대한 여부


  • ha=   ➜   수평 정렬. x, y 위치에 해당하는 부분을 어디로 정렬하는지. ( left, center, right )

  • va=   ➜   수직 정렬. x, y 위치에 해당하는 부분을 어디로 정렬하는지. ( top, center, bottom )


  • bbox=dict()   ➜   text 를 감싸는 box

    • dict(boxstyle=)   ➜   감싸는 box 형태 ( square, circle, larrow, darrow, rarrow )

    • dict(boxstyle=)   ➜   감싸는 box 테두리 형태 ( round, round4, sawtooth, roundtooth )

    • dict(facecolor=)   ➜   box 배경 색.

    • dict(ec=)   ➜   box 테두리 색

    • dict(alpha=)   ➜   box 투명도


  • ax.spine['position'].set_visible(False)   ➜   해당 position ( top, bottom, left, right ) 축 숨기기

    • ax.set(frame_on=False)   ➜   이 방식으로 모든 축 숨기기 가능

# Color

  • 범주형   ➜   채도와 광도는 유지하고, 색상의 변화만 준다.
  • 연속형   ➜   색조는 유지하고, 색의 밝기만 조정한다.

bar 에서는 color=, scatter 에서는 color=, c=, cmap=, vmin=, vmax= 등 과 같은 속성으로 색상 표현 방식을 정한다.

보기 좋게 색을 표현하는 방식이 다소 까다로워 공식문서 (opens new window)를 참고하며 사용 방식을 숙지하고, 여러 color palette 를 참고하여 적용시키는 연습을 하자.


# Facet

  Matplotlib 에서 Facet 이란 분할을 의미한다. 즉, 화면 상에 한 Figure 속 Ax 를 어떻게 배치할 것인가에 대한 내용이다. plt.subplot(), plt.figure() + fig.add_subplot(), plt.subplots() 과 같은 방식으로 배치하는 방식을 익혔지만 어디까지나 수평 방향 단일 공간 기준이다. figure 에서 단일 Ax 가 가로로 길게 차지하려면 어떻게 해야할까? 단일 Ax 가 세로로 길게 차지하려면 어떻게 해야할까?

  Grid Spec 을 활용한다면 numpy 의 slicing 방법처럼 fig.add_grid_spec() 이나 fig.subplot2gird() 에 적용시키면 된다. 그게 아니로 내부에 그린다고 한다면, ax.inset_axes() 나 make_axes_locatable(ax) 방식을 사용하면 되지만, 이 방법으로 원하는 형태를 그려내기가 어려워 권장하지 않는다. 또한, 전반적으로 다소 복잡한 배치를 가지는 구조를 원한다면, matplotlib 으로만 해결하려는 것이 아니라 단일 figure 들을 모아 figma 나 powerpoint 를 이용하면 어떨까?

  참고로 figure 를 저장하려면 fig.savefig('file_name', dpi=150) 코드를 실행시키면 된다. dpi 는 높을 수록 고해상도, 고용량 png 파일을 얻을 수 있다.


# Seaborn

  Seaborn 은 Matplotlib 기반 통계 시각화 라이브러리로써, Matplotlib 으로 커스텀이 가능할 뿐만 아니라, Matplotlib 으로만 구현할 때는 복잡했던 그래프를 쉽게 구현할 수 있다.

  우선 공통적으로 적용되는 파라미터가 있다. 이것을 알아본 다음, 그래프 그리는 함수를 차근차근 살펴보면 적용하는 방법이 무척이나 간단하다는 것을 알게 된다.

  • x   ➜   X 축 tick 에 적용시킬 feature name

  • y   ➜   Y 축 tick 에 적용시킬 feature name

  • data   ➜   보고자하는 대상 데이터

  • hue   ➜   두 개의 feature 와 더불어 추가로 보고자 하는 feature 속 category

    • hue_order   ➜   category 순서 정렬

  • order   ➜   feature 순서 정렬

  • ax   ➜   적용시킬 Ax

  • ETC   ➜   palette, color, saturate, ...


  • Categorical

    • Counter Plot   ➜   sns.counterplot(x='feature_name_1', y='feature_name_2', data=df, ax=ax)
    • Box Plot   ➜   sns.boxplot(x='feature_name_1', y='feature_name_2', data=df, ax=ax)
    • Violin Plot   ➜   sns.violinplot(x='feature_name_1', y='feature_name_2', data=df, ax=ax)
      • bw=   ➜   얼마나 자세히 표현할 지. 작을수록 세세 ( default = 0.2 )
      • cut=   ➜   양 끝 부분을 얼마나 자를지. ( cut=0 ➜ tick 범위만큼만 표현 )
      • inner=   ➜   내부를 어떻게 표현할 지. ( stick, quartile )
      • scale=   ➜   area, amount, width
      • split=   ➜   두 개의 category 를 동시에 비교 ( split=True )
    • Boxen Plot   ➜   sns.boxenplot(x='feature_name_1', y='feature_name_2', data=df, ax=ax)
    • Swarm Plot   ➜   sns.swarmplot(x='feature_name_1', y='feature_name_2', data=df, ax=ax)
    • Strip Plot   ➜   sns.stripplot(x='feature_name_1', y='feature_name_2', data=df, ax=ax)

  • Distribution

    • Hist Plot   ➜   sns.histplot(x='feature_name', data=df, ax=ax)
      • 히스토그램
      • binwidth=   ➜   몇 개의 tick 마다 막대 하나 그릴지.
      • bins=   ➜   막대 몇 개를 그릴지.
      • element=   ➜   step = 기본 hist 에서 테두리를 없앰 / poly = 다각형으로 바꿈
      • multiple=   ➜   stack = 어제 살펴본 stack 과 동일 / dodge = grouped 버전 / fill = percentage 버전
    • KDE Plot   ➜   sns.kdeplot(x='feature_name', data=df, ax=ax)
      • Kernel Density Estimate (연속 밀도 함수)
      • fill=True 로 설정하는 것이 일반적 ( 연속 밀도 함수임을 짐작하게 함 )
      • bw_method=   ➜   얼마나 세세하게 표현할 지. 작을수록 세세
      • cut
      • multiple
      • cumulative
    • ECDF Plot   ➜   sns.ecdfplot(x='feature_name', data=df, ax=ax)
      • 누적 밀도 함수
    • RUG Plot   ➜   sns.rugplot(x='feature_name', data=df, ax=ax)
    • Bivariate Distribution
      • 2개 이상 변수에 대해서 동시에 분포 확인
      • sns.histplot(x='feature_name', y='feature_name_2', data=df, ax=ax, cbar=False, bins=(10, 20))
      • sns.kdeplot(x='feature_name', y='feature_name_2', data=df, ax=ax, fill=True, bw_method=0.1)

  • Relation

    • Scatter Plot   ➜   sns.scatterplot(x='feature_name', y='feature_name_2', data=df, ax=ax, hue='feature_category')
    • Line Plot
    • Reg Plot

  • Correlation

    • Heat Map   ➜   sns.heatmap(df.corr(), ax=ax, vmin=-1, vmax=1, center=0, cmap='coolwarm', annot=True, fmt='.2f', linewidth=0.1, square=True)
    • Mask 를 만들어서 mask 속성에 할당하면 대칭이 되는 반대 삼각형을 제외하고 그릴 수 있다.

  • jointplot

    • 하나의 figure 를 기준으로 내부는 scatter 로 표현하고 외부는 hist 나 kde 로 표현

  • pairplot

    • 모든 feature 간의 관계 파악한다. 기본은 scatter 로 나타나고 대각 방향은 hist 로 나타난다. 단일 category 시 대각 방향은 hist, 복수 category 일 시 kde 로 나타난다.
    • 대칭되는 부분을 그리지 않으려면 corner=True 로 설정한다.

  • facetgrid

    • 다중 패널로 feature 간 관계 뿐만 아니라, feature 속 category 끼리의 관계도 파악할 수 있다.
    • catplot
    • displot
    • relplot
    • lmplot
