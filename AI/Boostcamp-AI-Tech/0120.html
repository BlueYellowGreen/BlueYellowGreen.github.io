# Vector

[1, 2] 처럼 표시되는 이 vector는 공간에서 한 점이라고 해석하기도 하고, 원점으로부터 상대적인 위치라고 해석하기도 한다. 여기서 원점으로부터 상대적인 위치로 해석하게 되면 노름(norm)이라는 개념이 나온다.

  Norm은 임의의 차원에 대해서 원점에서부터의 거리를 의미한다. 그 중에서도 - norm과 - norm 에 대해서 알아야 한다. 은 변화량의 절대값을 모두 더한 값이다. 이에 반해, 는 유클리드 거리로서 제곱 합의 평균이다.

  그러다보니, 은 절댓값의 합이라 에 비해 이상치에 대해 덜 민감하여 robust한 상황이나 Lasso Regression에 주로 적용된다. 반대로 는 에 비해 이상치에 민감하여, outlier 탐지하는 곳이나, 라플라스 근사, 그리고 Lasso Regression에 주로 적용된다.

# Matrix

  Matrix 에 대한 접근법을 vector 공간에서 사용되는 연산자라고 생각하자.
그렇다면 matrix product(행렬곱)은 vector를 다른 차원의 공간으로 보낸다고 볼 수 있다. 반대로, inverse matrix(역행렬)은 다른차원에 있던 vector를 되돌리는 것인 셈이다.

  문제는 역행렬이다. 기존에 배우기론, 역행렬은 row와 column의 개수가 같고 determinant가 0이 아닌 경우에만 구할 수 있다고 했다. 하지만 Tabular 데이터에 대해 선형 회귀를 진행(근사 함수의 계수를 찾기)하려고 하면 일반적으로는 row의 개수가 column의 개수보다 월등히 많다.

  이렇게 row와 column이 다른 경우에 대해서 유사역행렬(pseudo-inverse) 또는 무어-펜로즈(Moore-Penrose)를 이용하여 역행렬을 근사할 수 있다. 이러한 역행렬을 보통 에 대해 같이 +를 붙여 표현하며, 다음의 식으로 계산할 수 있다.

하지만, 우리는 앞으로 경사하강법을 사용한다.

# 경사하강법

  그라디언트 벡터 는 가장 빨리 증가하는 방향으로 흐른다. 그래서 각 변수에 대해 를 적용하면 최솟값을 구할 수 있다. 다만 여기서 무엇에 대한 최솟값을 구해야 하는지가 핵심이다. 우리는 학습을 통해 어떠한 데이터를 넣으면 최대한 정확한 예측값을 얻고 싶다. 여기서 예측을 하기 위해서는, 답에 가까울 것이라고 예상하는 함수나 분포에 대한 정보가 있어야 데이터를 입력해 계산할 수 있다. 이러한 것을 찾는 과정은, 주어진 데이터와 레이블(답)을 이용한다.

  SSE 는 오차의 제곱 합으로, 인과관계를 제대로 설명하는지 측정하기 위해 잔차의 합을 사용한다. 테스트 데이터를 이용한 계산에서는 자유도로 나누는 것이 아닌 으로 나눠야 한다. 알려지지 않은 복잡성은 MSE 계산에 들어갈 수 없기 때문이다.



  주어진 레이블과, 우리가 예상한 함수 혹은 분포에 데이터를 입력한 결과( )과의 차이( Loss )를 최소화하는 방향으로 진행하면, 예상했던 함수 혹은 분포가 점점 실제 상황에 가까워진다고 볼 수 있는 것이다.

  다만 여기에도 문제가 있다. 전체 데이터에 대해 계산을 진행하니, Loss function은 변하지 않는다. 그래서 학습 속도를 잘못 정하게 되면, local minimam에 빠지게 될 수도 있다. 이 뿐만 아니라, 학습 데이터가 많다면 메모리 부족으로 학습 자체를 진행하지 못할 수도 있다. 그래서 나온 것이 확률적 경사하강법이다.

# 확률적 경사하강법

  확률적 경사하강법(Stochastic Gradient Descent; SGD)은 모든 데이터에 대해 parameter를 업데이트하는 것이 아니라, 랜덤으로 데이터의 일부만 parameter 업데이트하는데 사용하는 것이다. 그래서 기존 경사하강법에서 Loss function의 고정으로 인해 local 언덕을 넘지 못할 상황에서도, 랜덤한 데이터 선택에 의해 Loss function이 변화하여 넘어갈 수도 있다.

  또한, 계산량 자체가 줄어 학습 속도적인 측면에서도 이점이다. 물론 SGD가 만능은 아니지만 실증적으로 더 낫다고 검증된 바가 있으니, 사용하지 않을 이유가 없다.

# Deep Learning

  딥러닝은 신경망을 깊게 쌓은 것이고, 신경망은 선형모델과 활성함수를 합성한 함수이다.
딥러닝으로서 의미있게 동작하기 위해서는 비선형 형태를 띄어야 하고, 따라서 활성함수는 비선형 함수를 사용해야 한다. Input 데이터로부터 을 계산하는 과정을 순전파(forward-propagation), Loss 를 최소화하는 방향으로 parameter 를 업데이트 하는 과정을 역전파(backpropagation)라고 한다.

  이론적으로는 2층의 신경망으로도 임의의 연속함수를 근사할 수 있지만, 층이 깊을수록 필요한 뉴런의 개수가 줄어든다. 하지만 무작정으로 깊게하면, gradient vanishing 문제가 발생한다.
