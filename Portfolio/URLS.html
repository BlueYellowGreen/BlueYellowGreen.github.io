<Android / Chrome Extention / Web>


Android 모바일 환경에서는 Urls App과 Chrome 인터넷을 이용하여 URL을 관리할 수 있습니다.

구글 확장프로그램을 통하여 간편하게 URL을 저장할 수 있습니다.

Urls 홈페이지에서 전체 URL들을 관리할 수 있습니다.



북마크하고 싶은 URL을 복사한 뒤,

Android / Chrome / Web 어디에서든지

붙여넣기 하면 됩니다.


URL을 붙여넣으면,

해당 URL html 내용을 바탕으로

태그들을 추천해줍니다.

또한, 카테고리를 지정함으로써

다른 URL을 추천받으실 수 있습니다.



URL 사용자를 초대해서 URL을 공유할 수 있고,
권한을 설정하여 편집 / 보기 권한을 부여할 수 있습니다.

그리고 북마크한 URL 내용에서 원하는 부분을
드래그하여 해당 URL의 메모로 저장할 수 있습니다.
