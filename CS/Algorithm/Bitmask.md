# Bitmask


**Bit** + **Mask** 라는 이름처럼,

Bit와의 동일성 유무를 판단하여 집합 연산에 주로 사용된다.

n 개의 원소를 가진 집합(배열)이 있을 때, 모든 조합의 경우는 $2^n-1$ 이다.
